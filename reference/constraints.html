Справочник ограничений валидации¶

Валидатор создан так, чтобы валидировать объекты с ограничениями В реальной жизни ограничением может быть: "Торт не должен сгореть". В Symfony, ограничения похожи: Они являются утверждениями того, что условие выполнено.

Поддерживаемые ограничения¶

Следующие ограничения по умолчанию доступны в Symfony:

Basic Constraints¶

These are the basic constraints: use them to assert very basic things about the value of properties or the return value of methods on your object.

  • NotBlank
  • Blank
  • NotNull
  • IsNull
  • IsTrue
  • IsFalse
  • Type
